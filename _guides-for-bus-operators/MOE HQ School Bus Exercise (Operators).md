<h1>MOE HQ School Bus Exercise (Operators)</h1>
<p>The MOE HQ School Bus Exercise is conducted annually, typically by the
end of February.</p>
<h3>Accessing the Data Exercise Page</h3>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/MOE HQ Data Exercise/OSB_Ops_40.png">
</div>
<p>To check your data entry progress for each of the schools you are serving:</p>
<ol data-tight="true" class="tight">
<li>
<p>Click on Data Exercise on the top navigation bar.</p>
</li>
<li>
<p>You will see the progress of data input for students, which includes any
errors in your data. Click on the provided link to access the relevant
page for making necessary edits.</p>
</li>
<li>
<p>Check the progress of data input for routes, along with any associated
errors. Click on the provided link to make edits as needed.</p>
</li>
<li>
<p>Review the progress of data input for your company profile and address
any errors by clicking on the provided link.</p>
</li>
</ol>
<hr>
<h3>Updating Missing Student's Information</h3>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/MOE HQ Data Exercise/OSB_Ops_41.png">
</div>
<p>Clicking on the Students link from the Data Exercise page leads you to
this page.</p>
<ol data-tight="true" class="tight">
<li>
<p>Ensure the <strong>Error/Incomplete Fields</strong> filter is on.</p>
</li>
<li>
<p>Fields that are incomplete or have errors are either shaded Red or Orange.
Tap on them to make edits.</p>
</li>
<li>
<p>Click on <strong>Save Changes</strong> to complete the modifications.</p>
</li>
</ol>
<p><strong>Note:</strong>
</p>
<ul data-tight="true" class="tight">
<li>
<p>If the page is empty with the Error/Incomplete Fields filter on, your
students have all the required information.</p>
</li>
<li>
<p>Bus Fare is the original bus fare without subsidy from the Financial Assistance
Scheme.</p>
</li>
<li>
<p>In the scenario that the student’s name cannot be found in the MOE database,
you can add the last 4 digits of the student’s NRIC/FIN to broaden the
search. Alternatively, check that the student’s name is properly spelled
and try saving again.</p>
</li>
</ul>
<h3>Declaring That You Are Unable to Obtain a Match Student by Name or NRIC</h3>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/MOE HQ Data Exercise/OSB_Ops_42.png">
</div>
<p><strong>[Important]</strong> If you have keyed in the last 4 digits of
the student’s NRIC/FIN and checked that the student’s name is correct,
but the system is still not able to find the student, you will have to
declare that the data is accurate to proceed.</p>
<h2>Completing the Submission for the MOE HQ Annual Exercise</h2>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/MOE HQ Data Exercise/OSB_Ops_43.png">
</div>
<p>Navigate back to Data Exercise by clicking on the button at the top right
of the page.</p>
<ol data-tight="true" class="tight">
<li>
<p>Click on Show more to view the progress for the Primary School you are
serving. Once there are no errors, you are ready to proceed.</p>
</li>
<li>
<p>Click on Notify School to bring up a pop-up if you are ready to notify
school AM via an email that your data is ready for submission to MOE HQ.</p>
</li>
</ol>
<p><strong>Note:</strong>
</p>
<ul data-tight="true" class="tight">
<li>
<p>Once you have notified school, you will receive a confirmation message
to note that you have successfully notified your school that data is ready
for submission.</p>
</li>
<li>
<p>Your school AM will get back to you should any information need to be
added/changed. There is no need to notify a second time should there be
any further changes.</p>
</li>
</ul>
