<h1>Creating Bus Routes (Manual Entry)</h1>
<p>This page provides step-by-step instructions for manually creating and
managing bus routes in OSB. The instructions cover tasks such as:</p>
<ul data-tight="true" class="tight">
<li>
<p>Creating new bus routes</p>
</li>
<li>
<p>Editing bus route information</p>
</li>
<li>
<p>Adding stops and students to a bus route</p>
</li>
<li>
<p>Add a new stop before or after an existing stop</p>
</li>
<li>
<p>Separate a student from a merged stop</p>
</li>
<li>
<p>Delete students/rows from the bus route</p>
</li>
<li>
<p>Change the order of the stops</p>
</li>
<li>
<p>Delete a route</p>
</li>
</ul>
<hr>
<h2>Creating New Bus Routes</h2>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Creating Bus Routes/OSB_Ops_4.png">
</div>
<p>Click on the <strong>Create Route</strong> button on the left sidebar and
fill in the fields on the form:</p>
<ol data-tight="true" class="tight">
<li>
<p><strong>School</strong>: What school does this route serve? If your school
is not on the dropdown, please contact the school’s Admin Manager to add
you to the system. 2.<strong>Pick Up/Drop Off</strong>: The route is Pick
Up if students are going to school, and Drop Off if the students are returning
home.</p>
</li>
<li>
<p><strong>Bus Route</strong>: The name of the bus route for your reference.
(Example: AM1, AM2, PM1, PM2)</p>
</li>
<li>
<p><strong>Bus No.</strong>: The bus no. will be reflected on the drivers’
attendance app. (Example: Bus 1, Bus 2)</p>
</li>
<li>
<p><strong>Vehicle Plate No.</strong>: The license plate number of your buses.</p>
</li>
<li>
<p><strong>Vehicle Capacity</strong>: The maximum number of students that
can fit in your bus.</p>
</li>
<li>
<p><strong>Start Time/Dismissal Time</strong>: If this is a pick-up route,
start time refers to the time at the first stop. If this is a drop-off
route, start time is when your bus leaves school.</p>
</li>
<li>
<p><strong>Driver/Attendant Mobile Number</strong>: We use this mobile number
for your staff to log into the attendance app, so make sure this is correct.
For foreign numbers, add the country code in front, e.g. +60.</p>
</li>
</ol>
<p>Press <strong>Save</strong> after you fill in the form. You will now see
your created bus route.</p>
<hr>
<h2>Edit Bus Route Information</h2>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Creating Bus Routes/OSB_Ops_6.png">
</div>
<ol data-tight="true" class="tight">
<li>
<p>Click on the <strong>Edit</strong> button to the right of the route information.</p>
</li>
<li>
<p>A form will appear with the route information pre-filled. Make changes
to the fields you desire.</p>
</li>
<li>
<p>Press the <strong>Save</strong> button on the form.</p>
</li>
</ol>
<hr>
<h2>Add Stops and Students to the Bus Route</h2>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Creating Bus Routes/OSB_Ops_7.png">
</div>
<p>Click on the <strong>+</strong> button below the table header. An empty
row will appear. Fill in the information of the first student in the cells
of the empty row:</p>
<ol data-tight="true" class="tight">
<li>
<p><strong>Name (Required)</strong>: Student name</p>
</li>
<li>
<p><strong>Contact (Required)</strong>: Contact number for the parent of
the student</p>
</li>
<li>
<p><strong>Contact 2</strong>: The second parent contact number for the student,
if any</p>
</li>
<li>
<p><strong>Stop Address (Required)</strong>: Address for the stop, e.g.,
565 Balestier Road</p>
</li>
<li>
<p><strong>Postal Code</strong>: Postal code for the stop</p>
</li>
<li>
<p><strong>Pick-Up/Drop-Off Time</strong>: Scheduled time at the stop</p>
</li>
<li>
<p><strong>Days taking Bus</strong>: Default is that students will be taking
the bus every day. Tap M, T, W, T, F to turn off the student if not taking
the bus on that day.</p>
</li>
</ol>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Creating Bus Routes/OSB_Ops_8.png">
</div>
<p>Press <strong>Save Changes</strong> button above the table.</p>
<p><strong>Note:</strong>
</p>
<ul data-tight="true" class="tight">
<li>
<p>You will only be able to save when you have filled in all required cells
and there are no red boxes around any of the cells.</p>
</li>
<li>
<p>A red box around a cell means that there is an error. Hover over the red
box for the error message and resolve the error accordingly.</p>
</li>
<li>
<p>An orange box around a cell means that there are additional actions necessary
for the MOE HQ data submission exercise. Hover over the orange box for
more information.</p>
</li>
</ul>
<hr>
<h2>Adding Students to a Stop</h2>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Creating Bus Routes/OSB_Ops_9.png">
</div>
<ol data-tight="true" class="tight">
<li>
<p>Hover over the first cell (Stop #) of the row. A checkbox and a menu button
will pop up.</p>
</li>
<li>
<p>Select <strong>Add student to stop</strong>. A new merged row will appear
below the current row.</p>
</li>
<li>
<p>Fill in the student’s name and contact.</p>
</li>
<li>
<p>Press <strong>Save Changes</strong>.</p>
</li>
</ol>
<hr>
<h2>Add a New Stop Before or After an Existing Stop</h2>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Creating Bus Routes/OSB_Ops_10.png">
</div>
<ol data-tight="true" class="tight">
<li>
<p>Hover over the first cell (Stop #) of the row. A checkbox and a menu button
will pop up.</p>
</li>
<li>
<p>Select <strong>Insert stop above</strong> in the menu if you want to add
a new stop before the current stop, or <strong>Insert stop below</strong> if
you want to add a new stop after the current stop. A new row will appear
before or after the current stop.</p>
</li>
<li>
<p>Fill in the student’s name and contact.</p>
</li>
<li>
<p>Press <strong>Save Changes</strong>.</p>
</li>
</ol>
<hr>
<h2>Separate a Student from a Merged Stop</h2>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Creating Bus Routes/OSB_Ops_11.png">
</div>
<p>If you have multiple students being picked up or dropped off at the same
stop, but one of the students is changing their stop, you can move the
student out of that stop. Follow the steps below.</p>
<ol data-tight="true" class="tight">
<li>
<p>Hover over the first cell (Stop #) of the row. A checkbox and a menu button
will pop up.</p>
</li>
<li>
<p>Select <strong>Split student out of stop</strong>. The student will now
be moved to a new row.</p>
</li>
<li>
<p>Fill in the stop details.</p>
</li>
<li>
<p>Press <strong>Save Changes</strong>.</p>
</li>
</ol>
<hr>
<h2>Delete Students/Rows from the Bus Route</h2>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Creating Bus Routes/OSB_Ops_12.png">
</div>
<ol data-tight="true" class="tight">
<li>
<p>Hover over the first cell (Stop #) of the row. A checkbox and a menu button
will pop up. Select the checkbox.</p>
</li>
<li>
<p>At the bottom of the screen, click on <strong>remove rows</strong>.</p>
</li>
<li>
<p>Click on <strong>remove rows</strong> again when the confirmation dialog
appears, and the row will be removed.</p>
</li>
<li>
<p>Press <strong>Save Changes</strong> to save your changes.</p>
</li>
</ol>
<hr>
<h2>Change the Order of the Stops</h2>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Creating Bus Routes/OSB_Ops_13.png">
</div>
<ol data-tight="true" class="tight">
<li>
<p>Click on the arrows to the left of the row.</p>
</li>
<li>
<p>The up arrow will move the stop up while the down arrow will move the
stop down.</p>
</li>
</ol>
<hr>
<h2>Delete a Route</h2>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Creating Bus Routes/OSB_Ops_14.png">
</div>
<ol data-tight="true" class="tight">
<li>
<p>Click on the <strong>Delete Route</strong> button to the right of the route
information.</p>
</li>
<li>
<p>A confirmation dialog will appear. Click on <strong>Delete</strong> to confirm.
The route will then be deleted.</p>
</li>
</ol>
