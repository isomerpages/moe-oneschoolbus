<h1>Upload Bus Routes - (Bulk Update)</h1>
<p>On this page, you will learn how to upload or update a lot of data at
one go in OneSchoolBus. This data comprises of your routes, driver, and
student information.</p>
<hr>
<h3>Accessing the Bulk Update page and templates</h3>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_15.png">
</div>
<p>From the top navigation menu:</p>
<ol data-tight="true" class="tight">
<li>
<p>Click on Routes</p>
</li>
<li>
<p>Click on Bulk Update.</p>
<p></p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_16.png">
</div>
</li>
</ol>
<p>You will see the page Bulk Upload. Select the template you would like
to download by following the following steps:</p>
<ol data-tight="true" class="tight">
<li>
<p><strong>Empty Template:</strong> Edit raw template and manually add in
your routes and student data.</p>
</li>
<li>
<p><strong>Pre-filled Template (Current Year):</strong> Edit template generated
with your existing routes and student data.</p>
</li>
<li>
<p><strong>Download button:</strong> Tap on the download button to receive
an excel template of your choice. You may proceed to open the excel template
to make changes.</p>
</li>
<li>
<p><strong>Next: Upload File:</strong> Once the excel template file is edited
and changed, you may proceed to upload the file by tapping on this button.</p>
</li>
</ol>
<hr>
<h3>Using the Excel template</h3>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_18.png">
</div>
<ol>
<li>
<p><strong>School Bus Template Instructions</strong>: The first sheet is
instructions on how to use the template. Please read it carefully before
filling in your data in the subsequent sheets.</p>
</li>
<li>
<p><strong>Start Inputting Data</strong>: You may tap on the tabs below to
start inputting bus route data into the template.</p>
</li>
</ol>
<p><strong>Note:</strong> Be prepared with a list of information such as Student
Name, Parent contact, pick up address and time, to bus route names, vehicle
plate number to driver and attendant name and contact number. You will
be required to input these details.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_19.png">
</div>
<ol data-tight="true" class="tight">
<li>
<p><strong>Description</strong>: Provides information on the required data
to be input into this excel template. Red fields are compulsory, while
orange fields are required only for the submission exercise sometime in
February annually.</p>
</li>
<li>
<p><strong>Filling in the Information</strong>: The fields are to be filled
in across in the direction displayed in the screenshot above.</p>
</li>
</ol>
<p><strong>Note:</strong> You can upload more than 3 buses with the excel
template. Although the downloaded blank school bus template only shows
3 buses, you may upload as many buses as you would like. Just duplicate
existing lists to add more.</p>
<p><strong>Note:</strong> You can duplicate lists for buses (IN) and (OUT).
Right click on the desired bus route and click move or copy.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_20.png">
</div>
<ol data-tight="true" class="tight">
<li>
<p><strong>Fill in bus route details on 1 row:</strong> Do note that you do
not need to drag route details for all student rows. You will only need
to complete the first row and leave the rest of the cells blank.</p>
</li>
<li>
<p><strong>Student Details</strong>: Each row starting from the student name
column represents a student, and all columns should be filled up to minimize
any missing data and upload issues later.</p>
</li>
</ol>
<p><strong>IMPORTANT:</strong> Remember to save your file after you have completed
the template!</p>
<p><strong>Note:</strong> If you are having difficulties with student class,
remember that this is an optional field! Feel free to leave it blank. Our
system may be able to autofill the value for you if it can find the student
in our system.</p>
<hr>
<h3>Upload File</h3>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_21.png">
</div>
<p>Once your template is ready, you may:</p>
<ol data-tight="true" class="tight">
<li>
<p>Click on Next: Upload file to upload your completed Excel Sheet.</p>
<p></p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_22.png">
</div>
</li>
<li>
<p>Click on the select drop-down to select the year you are uploading this
template for.</p>
</li>
</ol>
<p><strong>Note</strong>: If you selected the wrong year, the existing data
for the selected year will be overridden.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_23.png">
</div>
<p>After selecting the year,</p>
<ol data-tight="true" class="tight">
<li>
<p>You may either <strong>browse</strong> to look for your excel sheet or you
may <strong>drag and drop</strong> the file in the box indicated above.</p>
</li>
<li>
<p>Proceed to click <strong>Upload</strong> once the file is in.</p>
</li>
</ol>
<p>If you have uploaded your file successfully, congratulations! Find your
newly added information on the bus routes pages.</p>
<p>If not, donâ€™t worry! The error pages will indicate what went wrong.</p>
<hr>
<h3>Error Page 1</h3>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_24.png">
</div>
<p>The first error shown above indicates that the original headers of the
excel template have been changed, you will be required to redownload the
default template and re-enter the data without edits to the main headers.</p>
<ol data-tight="true" class="tight">
<li>
<p>Proceed to click <strong>Upload again</strong> once you have made the necessary
changes with the right main headers.</p>
</li>
</ol>
<hr>
<h3>Error Page 2</h3>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_25.png">
</div>
<p>The second error shown above indicates that there are either some incomplete
or unreadable fields within the excel template you have uploaded.</p>
<ol data-tight="true" class="tight">
<li>
<p>You are required to click download the <strong>File with error summary</strong> to
edit the changes and save the file.</p>
</li>
<li>
<p>Proceed to click <strong>Upload again</strong>, uploading the edited File
with error summary.</p>
</li>
</ol>
<p><strong>Note</strong>: The error summary page will point to you which
sheet and row where the error is located. Correct the highlighted cells
in your downloaded error file.</p>
<p><strong>Note</strong>: The red highlight on the error file will not go
away, but it does not affect your submission.</p>
<hr>
<h3>Copying this year data to next year</h3>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/Operator/Upload Bus Routes (Bulk Up)/OSB_Ops_17.png">
</div>
<p>At the end of the year, you may carry forward the data with the following
steps. Access the bulk upload page from the top navigation bar by clicking
on Routes and Bulk Update.</p>
<ol data-tight="true" class="tight">
<li>
<p>Click on <strong>Copy</strong>
</p>
</li>
<li>
<p>Click on <strong>Yes</strong> after the pop-up appears</p>
</li>
</ol>
<p><strong>Note</strong>: All P6 students will be removed, and all other
students with classes will be moved up by one level within the portal,
e.g., P1 to P2.</p>
